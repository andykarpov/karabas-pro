How to run ISDOS on Karabas-pro

1. Write image from is-dos_raw_hdd_image.zip to CF card (at least 64 mb size)
2. Put isdoshdd.$C somewere on SD card
   or/and flash ROMain_ramdisk_D_128_HDDBOOT.rom to ROMSET0, poweroff and poweron KARABAS-PRO
3. Enable NEMO IDE from on screen menu
3. Run isdoshdd.$C from SD card with File Simplorer or fatal or run HDD BOOT from BASIC128 menu
   if ROMain_ramdisk_D_128_HDDBOOT.rom is flashed into ROMSET 0
